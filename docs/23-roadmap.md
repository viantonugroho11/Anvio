# Roadmap

## Phase 0 — Foundation ✅
Monorepo, docs, core packages

## Phase 1 — MVP ✅
Runtime, Anthropic, Web Chat, Redis+PG memory

## Phase 2
MCP, Qdrant, approval workflow, admin UI

## Phase 3
Multi-provider, Telegram/Discord, workflows

## Phase 4
Multi-agent orchestration, K8s
